                                Newer versions of LFN Tools try to use the right character encoding for your drive. The least you want on your hard drive is a character set mess :-)

                                So LFN Tools check which character set you are using (mode con cp commands in your autoexec). LFN Tools have support build in for some widely used ones but not for all character sets. Especially asian and all double byte charsets are completely unsupported unfortunately. All supported character sets are compiled statically into the code. Because of the memory limitations under DOS it is not possible to support but a few character sets. Otherwise one would need to dynamically load the support in at runtime. But this would require a rewrite of the character set support code.

                                No. Development of LFN Tools has discontinued. If you would like CP 720 support you must first find somebody who is willing to continue to work on LFN Tools. The code is all openly available in CVS on sf.net

                                                • Re: Bug or HD broken? by Odi 30.11.03
                                                  > > ambiguous dpb!(50)
                                                  > > What does this mean? Thanks in advance. It is a nice program :-).

                                                  The very first sector (DPB = disk parameter block) of a FAT file system contains meta information about the file system such as how large it is and where the root directory is and so on. You can display that information with the LCHK tool.

                                                  LFN Tools check if the information presented in the DPB is consitent or if it just contains random values. This is to ensure that LFN Tools are only run on the supported file systems and that they do not operate on plain wrong information and possibly destroy the filesystem. So this is a security feature actually.

                                                  In your case it seems that the DPB is incorrect. You can check with LCHK. Maybe the drive is not FAT32 as you believe! You can try and repair the information with scandisk.

                                                                                                                                                    Well, this is the wrong tool for your problem.

                                                                                                                                                    Ambiguous DPB means that the data in the Disk Parameter Block (DPB, the 1st sector on that drive) is strange and probably corrupt. The message occurs if either:
                                                                                                                                                    - bytes per sector < 256 or > 65535 (normal is 512)
                                                                                                                                                    - bytes per sector is not a multiple of 256
                                                                                                                                                    - sectors per cluster is > 1024 or 0 (normal is 8 - 64)
                                                                                                                                                    - number of FATs exceeds 8 (normal is 2)

                                                                                                                                                    You do not want LFN Tools to read bad data off your disk.
                                                                                                                                                    You should use Norton Ghost or similar to mirror your bad disk to a stable disk (with all known good sectors). Then repair the filesystem (Norton Utilities or Scandisk).
                                                                                                                                                    Then use LFN Tools or Windows to rescue your data.

                                                                                                                                                                                                                                                                                                                    LCOPY allows a quick way to backup your Windows operating system.

                                                                                                                                                                                                                                                                                                                    First, clean up all the junk from your drive, such as
                                                                                                                                                                                                                                                                                                                    windows\temp\
                                                                                                                                                                                                                                                                                                                    windows\Temporary Internet Files\
                                                                                                                                                                                                                                                                                                                    the recycle bin
                                                                                                                                                                                                                                                                                                                    There are plenty of programs that do this well.

                                                                                                                                                                                                                                                                                                                    Next, reduce your swap file to say 20MB. Key...
                                                                                                                                                                                                                                                                                                                    Control Panel\System\Performance\Virtual Memory\
                                                                                                                                                                                                                                                                                                                    Make the minimium and maximium 20MB
                                                                                                                                                                                                                                                                                                                    Pass all the warnings and allow the reset.

                                                                                                                                                                                                                                                                                                                    Now make a folder, preferable on a different drive or partition from C: say
                                                                                                                                                                                                                                                                                                                    d:\winold\

                                                                                                                                                                                                                                                                                                                    Copy all folders you want to back up into winold, ie
                                                                                                                                                                                                                                                                                                                    c:\Program Files\ to d:\winold\Program Files\

                                                                                                                                                                                                                                                                                                                    You can also copy folders from other drives here now. After you restore windows, you'll restore them to those drives, so keep track of where they came from.

                                                                                                                                                                                                                                                                                                                    Do NOT copy the c:\windows or c: root files there yet. However make a folder d:\winold\windows\

                                                                                                                                                                                                                                                                                                                    For safety, put LCOPY.EXE on your boot disk as well as in d:\winold\

                                                                                                                                                                                                                                                                                                                    Now, here we go. With the boot disk in drive a: reset and boot to real-mode DOS. Make sure you can access your CD just in case.

                                                                                                                                                                                                                                                                                                                    Now key...
                                                                                                                                                                                                                                                                                                                    a:\lcopy c:\ d:\winold /a
                                                                                                                                                                                                                                                                                                                    a:\lcopy c:\windows\* d:\winold\windows /a /s

                                                                                                                                                                                                                                                                                                                    The first line copies all the c: root files (but no folders) to
                                                                                                                                                                                                                                                                                                                    d:\winold\

                                                                                                                                                                                                                                                                                                                    The second line copies the entire c:\windows\ folder and subfolders to
                                                                                                                                                                                                                                                                                                                    d:\winold\windows\

                                                                                                                                                                                                                                                                                                                    Now you can reboot back to windows and make sure everything was copied correctly. If you wish, you can burn d:\winold\ to a CD.

                                                                                                                                                                                                                                                                                                                    The best part of this is how easy it is to restore the operating system. Reboot to the boot disk and type (Gulp)
                                                                                                                                                                                                                                                                                                                    a:\FORMAT c:

                                                                                                                                                                                                                                                                                                                    Important: Once you do this, everything on your c: drive will be lost forever. You did triple check before doing it.

                                                                                                                                                                                                                                                                                                                    Now restore your original operating system with
                                                                                                                                                                                                                                                                                                                    a:\lcopy j:\winold\* c: /a /s

                                                                                                                                                                                                                                                                                                                    That's it. This one command does the entire restore! When you reboot, you'll be back to where you backed up.
